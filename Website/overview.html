System Overview

Architecture Diagram

The diagram represents our project's architecture. To enable the project to function properly, we'll need a Raspberry Pi, a camera module, a computer system, and a hand to operate the constructed model. As shown in the model, the Raspberry Pi is loaded with the model and then linked to the computer and camera module. The camera module uses the Raspberry Pi to capture the signals given by the hand and then helps the hand control the computer screen. In theÂ current model, we created two gestures: one to move the items on the screen and another to click the objects. We can add more gestures in the future. In the future, we will be able to add other gestures and entirely control the mouse functions, which could eventually replace the mouse.
